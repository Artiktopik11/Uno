# 🃏 UNO · P2P

Игра в УНО по сети без сервера.  
Peer-to-peer через PeerJS.

## 🚀 Как играть:
1. Один создаёт комнату → получает ID
2. Второй вводит ID → подключается
3. Играйте!

## 🔧 Технологии:
- PeerJS (WebRTC)
- Чистый JS + CSS

## 📱 Доступно:
- Компьютер
- Телефон (адаптив)

## 🎯 Фичи:
- ✅ Все карты УНО
- ✅ Wild + Wild Draw 4
- ✅ Выбор цвета
- ✅ Пас / Взять карту
